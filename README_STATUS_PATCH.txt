PATCH: статуси замовлення в особистому кабінеті

Що робить:
- показує статус замовлення в кабінеті користувача
- додає кольорові бейджі статусів
- показує список товарів у кожному замовленні

Статуси та кольори:
- Нове замовлення -> чорний
- В роботі -> жовтий
- Підтверджено -> блакитний
- Відправлено -> синій
- Завершено -> зелений
- Скасовано -> червоний

Що замінити:
- templates/account/dashboard.html

Важливо:
- якщо певне замовлення не відображається в кабінеті, перевір, чи воно створене з user_id поточного користувача
- якщо список товарів не показується, перевір, чи є зв’язок Order -> OrderItem у моделі:
  items = db.relationship("OrderItem", backref="order", lazy=True, cascade="all, delete-orphan")
